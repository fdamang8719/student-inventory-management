# Student Inventory Management System

## Project Overview
This project manages a list of inventory items. Each item has a name, a quantity in stock, and a price per unit. The application calculates the total value of the inventory.

## Files
- `inventory.py`: Defines a list of items with name, quantity, and price.
- `app.py`: Calculates and prints the total value of the inventory.
- `utils/helpers.py`: (Optional) Helper functions for advanced calculations.
